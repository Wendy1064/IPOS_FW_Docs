Detailed Description

FreeRTOS task and queue layer for master–slave UART communication.

This module creates the UART Master Task, which manages communication between the STM32 master and an external PLC/slave MCU. It acts as a middle layer between:

  • The Master Communication Link low-level UART + DMA driver, and
  • The Application Startup and Task Manager high-level task logic.

The UART Master Task initializes message queues for:

  • Acknowledgment frames (gAckQueue)
  • Data (read) frames (gDataQueue)

It spawns a FreeRTOS thread that continuously polls the Master Communication Link parser, processes incoming UART frames, and dispatches parsed messages to other tasks via message queues.

Responsibilities:

  • Initialize UART communication
  • Forward ACK and DATA frames from parser to queues
  • Handle ISR notifications via thread flags
  • Poll and drain protocol frames using master_link_poll

◆ UartMaster_StartTasks()

void UartMaster_StartTasks ( void * uart_handle)

Creates message queues and starts the UART Master task.

This function is called from Application Startup and Task Manager during startup. It creates the FreeRTOS message queues and starts the UART Master Task with high priority to maintain link timing.

Parameters
uart_handlePointer to initialized UART handle (e.g. &huart2)
Note
The created task runs under the name **"uart_master"** and priority osPriorityAboveNormal.

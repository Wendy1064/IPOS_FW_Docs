◆ App_BuildActiveStatusWord()

uint16_t App_BuildActiveStatusWord ( void )

Build composite active status word including current temperature.

Bit layout:

  • Bits [0..3]: Door, E-Stop, Key, BDO.
  • Bit [4]: Latch fault active.
  • Bit [5]: Temperature OK.
  • Bits [15..8]: Rounded temperature in °C.
Returns
16-bit encoded status word.

◆ Cond_LatchError()

uint8_t Cond_LatchError ( void )

Evaluate latch fault inputs and software flags.

Monitors all hardware latch error lines and software latch overrides. If a latch error occurs, disables laser, logs fault, and sets swLatchFaultActive = 1.

Returns
0 if no fault, 1 if latch fault active.

◆ Cond_RelayContactsMatch()

uint8_t Cond_RelayContactsMatch ( void )

Verify relay contact feedback consistency.

Checks NO1 and NC1 contact states after both relay coils are ON. A short grace period (50 ms) avoids false trips. If mismatch persists, logs a fault and forces latch error.

Returns
0 if contacts match (OK), 1 if mismatch detected (fault).

◆ vTaskInputs()

void vTaskInputs ( void * argument)

FreeRTOS task that scans inputs and checks safety conditions.

FreeRTOS task that scans inputs and checks safety conditions.

Performs:

  • 30 ms input debouncing.
  • Door/relay/latch rule checking.
  • Power-good monitoring (every 500 ms).
  • Automatic laser disable on power or latch fault.

FreeRTOS task that outputs event messages via USB.

The {\bfseries{Application Startup and Task Manager}} ({\ttfamily \doxylink{app__main_8c}{app\+\_\+main.\+c}}) is the central control module responsible for initializing all core firmware components and launching RTOS tasks. It ensures deterministic startup sequencing and safe coordination between the USB interface, input handler, flash logger, and communication tasks.

After Free\+RTOS initializes, {\ttfamily \doxylink{group__app__main_ga783607bdd6febf2b64a3f7871e1e9e4c}{App\+\_\+\+Start()}} creates all queues, event flags, and threads required for system operation.

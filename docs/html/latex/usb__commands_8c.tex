\doxysubsection{Detailed Description}
USB command interface for IPOS firmware. 

Handles user commands received via the USB CDC virtual COM port. Commands trigger debug actions, log operations, latch resets, and system tests.

This module parses ASCII text commands (e.\+g., "{}\+HELP"{}, "{}\+RESET"{}, "{}\+FLASH TEST"{}) and translates them into system events or control actions. Commands are posted as messages to the RTOS queue for asynchronous processing by the input/event system.

\begin{DoxyNote}{Note}
The module interacts primarily with the Input Handling and Flash Log subsystems. 
\end{DoxyNote}

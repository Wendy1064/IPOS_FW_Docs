This page documents the {\bfseries{Application Data Instances (ADIs)}} and {\bfseries{Process Data Mapping}} used by the IPOS Anybus M40 firmware. ~\newline
 Each ADI defines a logical variable visible to the {\bfseries{Profinet PLC}} and synchronised with the {\bfseries{STM32 master controller}} over UART3.

The firmware implements direct mirroring of PORTA/\+B/C digital I/O and several custom status words.

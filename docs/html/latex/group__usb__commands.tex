Processes incoming USB commands from the CDC interface. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em cmd} & Null-\/terminated ASCII command string.\\
\hline
\end{DoxyParams}
Recognized commands trigger specific actions such as\+:
\begin{DoxyItemize}
\item {\bfseries{HELP}} — Lists all available commands.
\item {\bfseries{VERBOSE ON/\+OFF}} — Enables or disables detailed debug output.
\item {\bfseries{STATUS DEBUG}} — Requests current input status report.
\item {\bfseries{TRUPULSE}} — Displays Trupulse monitor pin states.
\item {\bfseries{BDO TEMP}} — Displays current thermocouple reading.
\item {\bfseries{LOG DUMP / LOG ERASE}} — Manages non-\/volatile event logs.
\item {\bfseries{FLASH TEST / FLASH ID / FLASH STATUS}} — Tests or queries SPI flash.
\item {\bfseries{RESET}} — Issues a software latch reset event.
\end{DoxyItemize}

Unrecognized commands print an error message and a hint to use {\ttfamily HELP}.

\begin{DoxyNote}{Note}
Commands are parsed case-\/insensitively. 
\end{DoxyNote}

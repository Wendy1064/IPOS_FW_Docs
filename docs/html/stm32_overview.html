IPOS STM32 Housekeeping Overview

The Housekeeping Controller firmware provides all low-level control, safety, and monitoring functions for the IPOS laser system.
It runs on an STM32F4 MCU under FreeRTOS, coordinating interlocks, relay outputs, and temperature feedback via the SP-ICE-3 laser controller.

Responsibilities

  • Debounce and monitor all safety interlocks
  • Manage relay and output control
  • Process serial commands via USB
  • Exchange cyclic I/O data with Anybus M40 (UART3)
  • Provide diagnostic and debug status feedback
